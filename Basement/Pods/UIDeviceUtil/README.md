# UIDeviceUtil

*UIDeviceUtil* is a simple lib to know the exact type of the device instead of just iPhone or iPad.

  - Easy to use
  - Light weight

### Version
1.0.2

### Installation
Add this in your pod file
```sh
pod 'UIDeviceUtil', '1.0.2'
```
